# 第一次作业

## 数据说明

本次作业收集了自己一周以来（9.21-9.27）每天三餐主食的种类。

本次作业的数据收集中发现的问题及感想主要是：
1. 收集数据的过程比较繁琐，经常会忘记记录。
2. 若不进行长期的数据收集，无法看出明显的收集意义。
3. 多少会为了数据多样而影响自己的饮食选择。
