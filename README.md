- 如今我们已经来到了2019年的最后一个月。截至目前，无论是国内还是国外，在北半球的各个省市陆续入冬之后，各大音乐节品牌在2019年的活动也基本落下了帷幕，大家可以尽可能对来年的活动展开畅想与期待。这一年的音乐节和往年一样，经典的音乐节如期而至，并且其中的一些品牌不断前往更多新的城市供人们狂欢；全新的音乐节频繁涌现，尽管有一些只是为了商业的宣传，规模不大也很可能不具有延续性。总之，2019年的音乐节市场整体依然呈现稳步发展的状态。
- 现代音乐节的历史对于音乐爱好者，特别是独立音乐和摇滚乐的粉丝们来说可能比较熟悉。1969年的伍德斯托克（Woodstock）音乐节算是被普遍看作现代音乐节的开端。与其它多数音乐节不同，伍德斯托克音乐节并非一年一度的活动。在1969年第一届成功举办之后，主办方又分别于1994年、1999年举行了第二、三届，主题是“和平、反战、博爱、平等”，活动规模与演出阵容史无前例。不过，本应是正值伍德斯托克五十周年的2019年，计划好的音乐节却取消了。在预算严重超标、场地谈不拢、后勤安全也无保障的情况下，尽管前期主办方和金主已经投入了巨大的时间和精力成本，但也无法相信这次能够办出对得起“伍德斯托克”名号的音乐节——人们无法保证场地的清洁卫生，更无法保证艺人、合作伙伴和观众们的安全。因此，音乐节的“老祖宗”也只能在官网上颇具豪情壮志地打出一句“虽然音乐节被取消，但伍德斯托克精神将长存！”
- 不过，除了神一般存在的伍德斯托克，还是有很多经典的“等效替代品”供人们在冬季来临之前狂欢享乐的。其中，美国的Coachella音乐节、英国的Glastonbury音乐节，以及日本的FujiRock富士音乐节等都是海外著名的老牌音乐节。Coachella音乐节是美国目前规模最大的音乐节，诞生于1999年，在美国加利福尼亚州印第奥市举行。一般为期三天，场地处在沙漠之中，并至今保持着一年一度不间断的频率。
- Glastonbury音乐节是另一个历史悠久又规模庞大的音乐节。始于1970年的它是目前英国本土乃至全世界规模最大的室外音乐节。首届Glastonbury音乐节在农场中举行，观众仅需要支付1英镑就可以获得门票，并可以免费享用农场提供的新鲜牛奶。虽然没有严格保持着一年一度，在近五十年来的历史中，Glastonbury音乐节的规模依然不断扩大，门票价格也随着观众的人数不断上涨。（图2为Glastonbury音乐节历年的票价和观众数量的统计图）
![pic2](https://github.com/MeatbunBigking/Homework/blob/master/ticketattend.png)
- 再来看看我们的邻居日本。FujiRock富士音乐节是亚洲最受欢迎的音乐节。第一届富士音乐节举行于1997年，每一年几乎都在7月的最后一个周末举办，一般为期三天。2019年，eplus票务正式承担了海外票务的代理工作，这也大大方便了国内乐迷购买门票并前去观看。作为全亚洲最大的音乐节，富士音乐节在保证邀请全球大牌音乐人登场的同时，也为日本本土的音乐人提供了足够的演出舞台。除此之外，日本的Summer Sonic、Rock In Japan等音乐节也十分著名。
- 反观国内，几大音乐节品牌早已经如雷贯耳。迷笛音乐节算是室外音乐节在国内的发源地。诞生于2000年，迷笛音乐节将足迹带到了许多大小城市，并打造了“迷笛音乐学校”的品牌。当然，近些年来最为炙手可热的品牌还要数摩登天空旗下的草莓音乐节。2009年第一届草莓音乐节以来，摩登天空不仅将他们的音乐节带到了各大省份，也逐步将草莓音乐节，以及公司旗下的音乐人带到了海外。如今，草莓音乐节甚至计划对抗寒冷的冬季，在哈尔滨举办了“草莓恒温派对”。当然这种特别企划的活动并不能算进室外音乐节的范畴。同时，草莓音乐节还率先推出了丰富的版块，为电子音乐、说唱音乐和儿童等不同群体打造了专门的音乐节。
![pic3](https://github.com/MeatbunBigking/Homework/blob/master/caomeiyyj.png)
- 香港的Clockenflap音乐节不仅是中国最具“国际范儿”的音乐节，也是最受国外观众肯定的活动。2008年第一届Clockenflap在数码港举办。不光是音乐，Clockenflap致力于汇聚各种形式的艺术以及艺术家于一身，打造前卫的文化。不过，2019年的Clockenflap音乐节出于一些原因并未如期举办。
- 另外，太合麦田主办的“麦田音乐节”、天漠音乐节、西湖音乐节，以及来自台湾的简单生活节，也都为国内观众们提供了打发闲暇时光的好去处。
- 以上就是海内外音乐节的简单汇总。无论如何，音乐节都是能够跨越年龄、地域和民族，将各种形式的艺术汇集在一起，并且让人们在其中享乐的盛会。我们有幸看到，在各种品牌的音乐节不断登场、粉丝年龄段不断更新的同时，这些“老字号”的节日虽然偶尔会中断，品牌本身和背后的故事却不会消失。2019年即将结束，我们也许错过了这一年所有的音乐节，却总有下一个年头值得期待。

- 题目：《青少年犯罪的背后————不可忽视的心理建设》
- 10月份发生的“大连女童之死”案引发了国内对于未成年人保护和未成年人犯罪行为的又一次大规模讨论。遇害的女童年仅十岁，而凶手更是只有13岁。除了谴责凶手的残忍和思考法律制度问题等常规角度以外，作为以未成年人为事件中心的案件，我认为从凶手的角度出发，可以研究未成年人刑事案件与心理医疗条件的关系。
- 我先从国家数据中找到了2008-2017年青少年刑事案件的数量表，接着通过镝数的线上图表生成工具，制作了该表格的折线统计图。（如下图）
![刑事案件](https://github.com/MeatbunBigking/Homework/blob/master/xingshianjian.jpg)
- 从中我们可以发现，尽管青少年刑事案件从2008年以来整体的数量呈下降趋势，但整体数量仍然很高。其中，在2016-2017年的时间里，嫌疑人不满18岁的刑事案件的占比甚至呈增大趋势。案件整体的数量下降不仅得益于我国在青少年教育工作及法律方面的重视，更与近些年来未成年人口数量不断下降，以及针对未成年人案件“少捕、慎诉、附条件不起诉”的原则有关。2003 年至 2015 年，全国检察机关经审查不批准逮捕未成年犯罪嫌疑人 16 万余人，不捕率为 14.8% ；不起诉 5 万余人，不诉率为 4.4% 。而在 2017 年前 11 个月，全国对未成年犯罪嫌疑人不捕、不诉率已上升至 33.4% 和 18.4% （数据来源：最高检）。
- 心理健康与青少年犯罪的关系引发过长久且大规模的讨论。一般而言，关注青少年心理健康与否和青少年的犯罪行为数量存在一定关系。通过世界卫生组织官网提供的数据，我利用Tableau制作出了截止2017年世界各国心理医疗设施的配备情况图（见链接）。从中我们可以看出，对比世界其他发达国家，我国作为一个人口基数较大的国家，心理医疗设施并不完备。
<https://public.tableau.com/profile/li.jia.hao#!/vizhome/13372/1?publish=yes>
- 尽管通过研究心理设施分布的角度并不能直接影响未成年人保护和未成年人犯罪等相关问题的解决，也不能直接控制犯罪行为的发生频率，但是心理健康关注的不足，多少也算是引发犯罪的原因之一。通过这一角度，希望可以引起读者的重视，对青少年犯罪行为防患于未然，及时关注心理健康。
- 使用工具：Tableau、镝数
- 数据来源：国家数据、《司法大数据专题报告》、世界卫生组织
- 存在问题：许多有关未成年人犯罪的报道均属于禁区，获得详细的数据非常困难。

- 题目：《“麦辣鸡腿堡”的包装进化史》
- 在大城市面对垃圾分类问题纷纷跃跃欲试的同时，以麦当劳为首的国际快餐品牌也在餐厅里开始着食品包装垃圾的分类。在北京的麦当劳餐厅，按照包装材质分类的垃圾桶已经投入了使用。抛开尚未开始实施的大规模全民垃圾分类政策，麦当劳产品中的食物包装问题同样引发了消费者的好奇心。

![麦辣鸡腿堡](https://github.com/MeatbunBigking/Homework/blob/master/mailajituibao.jpg)
- 在这幅图片中，我通过Kaggle和麦当劳官网两个平台得到了数据。首先我介绍了北京目前的麦当劳门店数量情况，其次根据官网给出的“麦辣鸡腿堡每秒钟销售四个”的数据，自行计算得出了北京地区麦当劳每小时卖出的麦辣鸡腿堡数量，进而可以基本得出“一小时内北京地区麦当劳售卖的麦辣鸡腿堡”产生的汉堡包装纸数量。其次，在图片的下半部分，我通过查阅好奇心日报、麦当劳官网的数据和资料，了解到了中国麦当劳在汉堡包装材料上的变化情况，用图片更直观地体现了出来。从中我们可以看到，虽然每天麦当劳的食物包装产生的垃圾很多，但商家在包装的改良上依然费尽心思，不断推陈出新。
- 食物包装所产生的垃圾算是垃圾的一种，然而既然是广义的“垃圾”二字，被人戏称“垃圾食品”的，以麦当劳为代表的西式快餐产品也可以作为“垃圾”的一个新的角度。我从Kaggle上选取了麦当劳8种主要汉堡产品的部分营养成分含量，并制作成了条形图，配上汉堡的图例，简单表述出“一个汉堡含有多少脂肪/糖分/热量”的信息，供读者参考。
![ingredients](https://github.com/MeatbunBigking/Homework/blob/master/ingredients%20of%20MCD.jpg)
- 显然，垃圾食品并不止麦当劳这一种，食品包装的垃圾也并不只是麦辣鸡腿堡一样产品所产生的。所以，此次可视化作品仅作为某一种食品包装问题的展示样例，借此以小见大，让读者关注快餐的包装问题和演变的故事。

- 数据来源：好奇心日报、麦当劳官网、Kaggle
- 思路：由“垃圾”二字联想到“垃圾食品”，既分析垃圾食品的包装问题，也扩展到垃圾食品本身的问题。
- 工具：Tableau、Photoshop
